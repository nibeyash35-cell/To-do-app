# To-Do Flow

A beautiful and interactive To-Do List App built using HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Mark tasks as completed
- Edit tasks
- Delete tasks
- Filter tasks by All / Active / Completed
- Clear completed tasks
- Light/Dark theme toggle
- Local storage support
- Responsive and aesthetic UI

## Tech Stack

- HTML
- CSS
- JavaScript
